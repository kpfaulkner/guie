# DESIGN

## GOAL

Make a Golang library that can be used to make cross platform applications. 
This should be a user interface library/framework that provides things such as 
windows, widgets, events, scrolling, menus, lists, dropdown combos etc.

As a graphical basis it should use the EBiten (github.com/hajimehoshi/ebiten/v2) library.

Features that should be available:

- windows
- buttons
- scroll bars
- menus
- lists
- checkboxes
- radio buttons
- events
- themes
