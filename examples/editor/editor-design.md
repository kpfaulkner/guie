# Design

## Purpose

- Make a general purpose text editor. 
- Have menu bar with the menus file, edit and help.
- Under file menu, should have open, save and quit
- open should show a modal dialog to specify which file to open, then loads file
  populates contents into the editor
- save option in menu should save contents to file
- quit should quit the application
- edit menu should have 'find', 'replace' entries.
- 'find' should allow searching of text in editor
- 'replace' should allow replacing of one string with another within the editor
